# 主要功能

-   查询疫情相关的新闻资讯（目前新闻源为`央视新闻app`）
-   查询疫情的基本数据，如累计确诊、现存确诊等信息（目前主要是江苏省的数据）
-   查询自费核酸检测点信息（目前主要是江苏省的数据）
-   查询风险地区（目前主要是江苏省的数据）
-   核酸检测结果、通信行程卡查询（跳转`国家政务服务平台`小程序）
